# compact_LTM4607_PCB
A compact, 98% efficient, non-isolated DC-DC converter board based around the LTM4607
Input: 4.5-36V
Output: 0.8-24V, 5A (boost mode), 10A (buck mode)

## Version 1
Stand-off mounting, solderable wire connections

## Version 2
Card-edge style, compacted
